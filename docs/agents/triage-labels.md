# 트리아지 라벨

스킬들은 다섯 개의 표준 트리아지 역할을 기준으로 동작한다. 이 파일은 그 역할들을 이 저장소의 실제 라벨 문자열에 매핑한다.

| mattpocock/skills 라벨 | 우리 트래커 라벨   | 의미                                    |
| ---------------------- | ------------------ | --------------------------------------- |
| `needs-triage`         | `needs-triage`     | 메인테이너가 평가해야 함                |
| `needs-info`           | `needs-info`       | 리포터의 추가 정보를 기다리는 중        |
| `ready-for-agent`      | `ready-for-agent`  | 완전히 명세화됨, AFK 에이전트가 처리 가능 |
| `ready-for-human`      | `ready-for-human`  | 사람이 직접 구현해야 함                  |
| `wontfix`              | `wontfix`          | 처리하지 않음                            |

스킬이 어떤 역할을 언급하면(예: "AFK-ready 트리아지 라벨 적용"), 이 표의 우측 라벨 문자열을 사용한다.

실제 사용하는 라벨 어휘가 다르면 오른쪽 열만 수정한다.
